# Button Icons

There may be times where you simply want to convey the context of a button through a simple icon. In this case we use a slightly different component to that of normal buttons that provides more appropriate styling and accessibility.

Icon buttons are often found in nav bars and toolbars.

## Button Icon Variants

`<ButtonIcon />` components take in a `variant` property that defines the way the button should look and behave across the platform.

<Canvas>
  <ButtonIcon variant='primary' icon={<CloseIcon />} />
  <ButtonIcon variant='outlined' icon={<CloseIcon />} />
  <ButtonIcon variant='secondary' icon={<CloseIcon />} />
  <ButtonIcon variant='subtle' icon={<CloseIcon />} />
  <ButtonIcon variant='link' icon={<CloseIcon />} />
  <ButtonIcon variant='destructive' icon={<CloseIcon />} />
  <ButtonIcon variant='destructive-outlined' icon={<CloseIcon />} />
</Canvas>

## Button Icon Sizes

Not all buttons are created equal, and sometimes you may want to have a smaller or larger button depending on the context in which it's being used. For this reason you can pass a `size` property which will set the proper height and padding as necessary. By default buttons size to `md`.

<Canvas>
  <ButtonIcon size='xs' icon={<ArrowRightIcon />} />
  <ButtonIcon size='sm' icon={<ArrowRightIcon />} />
  <ButtonIcon size='md' icon={<ArrowRightIcon />} />
  <ButtonIcon size='lg' icon={<ArrowRightIcon />} />
</Canvas>
